# FortuMars Product Website

A modern, responsive product website for FortuMars AI Technologies, designed with a clean, high-end "Google Store" aesthetic.

## Features
- **Light Theme & Minimalist Design**: Clean typography, soft shadows, and white-space utilization.
- **Product Pages**: Detailed pages for Business Cloud, Data Analytics, and Consulting.
- **Responsive Navigation**: Fully functional on mobile and desktop.
- **Smooth Animations**: Intersection Observer API used for scroll-reveal effects.

## Technologies
- HTML5
- CSS3 (Custom Properties / Variables)
- Vanilla JavaScript
- FontAwesome Icons
